 # spark-nasa-request-server

 ## Descrição
 
 Analise dos arquivos de log das solicitações (requests) efetuados ao servidor NASA Kennedy Center.
 
 ### Arquivos
 
 Os arquivos access_log_Jul95 e access_log_Aug95, devido ao seu tamanho não foram  acrescentados, mas pdoem ser baixados na url oficial: (http://ita.ee.lbl.gov/html/contrib/NASA-HTTP.html)

 Todos os arquivos devem ser adicionados no diretório (data)

 ## Tecnologias

 - Apache Spark
 - Java 8
 - Apache Maven 3

 ## Execução
 
 Para realizar a execução da aplicação basta rodar a classe:
 
 ```
 br.com.projeto.logic.ExecuteProcess
 ```
